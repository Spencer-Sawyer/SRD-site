Investigate

A deep and thorough look into an area, object, and other things of the sort. This skill allows you to spend the time to find things that aren't so obvious to a quick glance. Doing so takes time though, ten minutes for a full investigation of a room. Others with skills in investigate can assist, though only one investigate check can be made for the whole group when this is done.
