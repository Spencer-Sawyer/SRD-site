Spell Focus [Magic Feat]

Prerequisite: Able to cast the spells selected

Benefit: Select a number of spells equal to your character's primary mental attribute. These spells are cast as if 1 CL Higher. If the damage/healing die limit is reached, for the selected spells this will grant another damage/healing die over the limit.

Special: This feat can be taken multiple times, each time it would apply to different spells
